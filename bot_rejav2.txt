# Project: "Sud Digital Assistant" Telegram Bot (Extension of "Sud Qo'llanma" App)
# Context: We are building an ecosystem to improve the DIGITAL COMPETENCE of Court Staff in Uzbekistan. This bot is NOT a lawyer; it is an IT Mentor and System Guide.

# 🎯 OBJECTIVE:
Build a secure, AI-powered Telegram bot that helps court employees use digital systems (E-SUD, E-XAT, Video conferencing) and solves technical issues. It must sync XP/Progress with the main Flutter App via Firebase.

# 🛠️ TECH STACK:
- Language: Python 3.11+ (Aiogram 3.x)
- Database: Firebase Firestore (Shared with Flutter App)
- AI: Gemini 1.5 Flash (System Prompt: "You are an expert IT Support for Court Systems")

# 🧠 CORE LOGIC & FEATURES:

## 1. Auth & Sync (The Bridge)
- **Deep Linking:** Command `/start` checks if user exists in Firestore (`users` collection).
- **Contact Match:** Securely verify phone number to link Telegram ID to the App Profile.
- **Sync Message:** "Hurmatli {name}, sizning ilovadagi darajangiz: {level}. Keling, raqamli bilimlarni oshiramiz!"

## 2. AI "HelpDesk" (RAG for Manuals)
- **Problem:** Users struggle with software (e.g., "How to upload files?").
- **Solution:**
    - User sends text.
    - Bot searches Firestore `articles` (Topic: Guides, Manuals).
    - AI generates a technical solution based on retrieved docs.
    - **Tone:** Professional, Technical but Simple, Supportive.

## 3. "Digital Quiz" Module (Gamification)
- **Content:** Questions about Information Security, Computer Literacy, and Court Systems Usage.
- **Action:**
    - User answers correctly.
    - **ATOMIC WRITE:** `firestore.collection('users').doc(uid).update({'xp': firestore.Increment(10)})`.
    - Notification: "+10 XP! (Sinxronlandi ✅)".
